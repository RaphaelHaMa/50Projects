# 50Projects
## 50 Projects in 50 Days
This repository contains 50 pages that I will create over the course of the next 50 days. The code for these pages is based on the Udemy course '50 Projects In 50 Days - HTML, CSS & JavaScript' by Brad Traversy & Florin Pop.
In the course students create web pages with HTML, CSS and JavaScript. This project is based on razor pages and C# code, but the design and functionality of the pages is based on the course. The course is a great way to learn web development and I highly recommend it.
